# 🌱 Pikmin S2 Map 使用教學

**網址：** https://agent-mouses.github.io/pikmin-s2-map/

用手機瀏覽器直接開啟，不需要安裝任何 App。

---

## 🗺️ 地圖頁（Map）

### 基本操作
1. 打開網頁，預設顯示台北地區的 **S2 Cell Level 17** 網格
2. 每個格子大約 **150m × 150m**，就是皮克敏 Bloom 判定飾品掉落的最小單位
3. 用手指**拖曳**移動地圖，**雙指縮放**調整大小

### 🎯 GPS 定位
1. 點右下角的 **🎯 按鈕**
2. 允許瀏覽器取用位置
3. 地圖會飛到你的位置，你所在的格子會用**青色**高亮
4. 再按一次 🎯 關閉定位

### 📐 切換 Level
- 上方滑桿可以切換 **Level 12～20**
- **Level 17** = 皮克敏 Bloom 用的格子大小（最常用）
- **Level 15** = Pokemon GO 的 EX 團體戰格子
- Level 越大格子越小，越小格子越大

### 🔲 點擊格子
點任何一個格子，底部會彈出資訊面板：
- **Cell ID** — 點擊可複製（分享給朋友用）
- **Level** — 目前顯示的等級
- **Center** — 格子中心座標

### ✅ 標記格子
在格子資訊面板裡有三個標記按鈕：

| 標記 | 用途 | 顏色 |
|------|------|------|
| ✅ farmed | 已經刷過的格子 | 綠色 |
| ⏳ cooldown | CD 冷卻中的格子 | 黃色 |
| ⭐ bookmark | 想去但還沒去的格子 | 粉色 |

- 點一次標記，再點一次取消
- 標記會存在手機裡（localStorage），關掉網頁不會消失

### 📍 顯示 POI（興趣點）
1. 點右側的 **📍 按鈕**開啟 POI 圖層
2. 會從 OpenStreetMap 查詢附近的餐廳、咖啡廳、公園、車站等
3. 每個 POI 會顯示對應的**飾品類型 emoji**（🍽️ 餐廳、☕ 咖啡廳、🍀 公園...）
4. 點 POI 可以看名稱和飾品類型
5. 再按一次 📍 關閉

> ⚠️ POI 查詢需要網路，第一次可能要等幾秒

## 📖 飾品圖鑑頁（Decor）

### 瀏覽飾品
1. 點底部的 **📖 Decor** 分頁
2. 所有 96 種飾品分類都在這裡
3. 點分類名稱展開，看到所有變體和皮克敏類型

### 標記收藏
- 每個皮克敏圖示是一個按鈕
- **點一下** = 標記為已收集（綠色邊框）
- **再點一下** = 取消收集
- 上方會顯示每個分類的收集進度（例如 `3/16 (19%)`）

### 篩選和搜尋
- **下拉選單**：篩選類型（Location / Event / Roadside / Weather / Regional）
- **搜尋框**：輸入名稱搜尋（支援中英文）

---

## 📊 統計頁（Stats）

1. 點底部的 **📊 Stats** 分頁
2. 最上方顯示**總收集進度百分比**
3. 下方按分類列出每個飾品的收集狀態
4. 也會顯示你標記了多少格子（farmed / cooldown / bookmark）

### 💾 備份與還原
- **📤 Export Data** — 匯出所有收藏和標記資料為 JSON 檔案
- **📥 Import Data** — 從 JSON 檔案匯入（換手機時用）

> 💡 建議定期匯出備份，因為資料存在瀏覽器裡，清除瀏覽器資料會遺失

## 🎮 實戰刷盆流程

### 出門前
1. 打開地圖，GPS 定位到你的位置
2. 開啟 POI 圖層，看附近有哪些飾品類型的格子
3. 找一個有多個格子的大純點（公園、百貨公司）
4. 用 ⭐ bookmark 標記想去的格子

### 到現場
1. GPS 確認你在哪個格子裡
2. 在格子裡走 **1,000～3,000 步**觸發掉落
3. 刷完一格標記 ✅ farmed
4. 如果格子進入 CD，標記 ⏳ cooldown
5. 移動到下一個格子繼續

### 回家後
1. 到 Stats 頁看今天的成果
2. 到 Decor 頁標記新收集到的飾品
3. Export 備份一下資料

## ❓ 常見問題

**Q: 格子顯示「Too many cells. Zoom in.」？**
A: 畫面範圍太大，放大地圖或降低 Level 就好。

**Q: POI 按鈕按了沒反應？**
A: Overpass API 有速率限制，等幾秒再試。如果一直沒反應可能是伺服器忙碌。

**Q: 換手機怎麼辦？**
A: 在 Stats 頁用 Export 匯出 JSON，在新手機用 Import 匯入。

**Q: 可以離線用嗎？**
A: 地圖和 POI 需要網路，但收藏資料存在本地，離線時可以看 Decor 和 Stats（如果之前有載入過）。

**Q: 這個工具跟 Pik-Tool 有什麼不同？**
A: Pik-Tool 需要帳號（Supabase 後端），我們的完全不需要帳號，資料存在你手機裡。我們多了 S2 Cell 多 Level 切換和格子標記功能。
