🌱 皮克敏 S2 地圖

—
L17 Pikmin (~150m)

🌱 使用教學

用手機瀏覽器直接開啟，不需要安裝 App。

🗺️ 地圖頁

🎯 GPS 定位

點右下角 🎯 → 允許定位 → 你所在的格子會用青色高亮。再按一次關閉。

📐 切換 Level

上方滑桿切換 Level 12～20。

  • Level 17 = 皮克敏 Bloom 的格子（最常用）
  • Level 15 = Pokemon GO EX 團體戰格子
  • 數字越大格子越小

🔲 點擊格子

點任何格子 → 底部彈出 Cell ID、Level、座標。點 Cell ID 可複製。

✅ 標記格子

✅ farmed已刷過的格子綠色
⏳ cooldownCD 冷卻中黃色
⭐ bookmark想去但還沒去粉色

點一次標記，再點一次取消。標記存在手機裡，關網頁不會消失。

📍 顯示 POI

點 📍 開啟 → 從 OpenStreetMap 查詢附近餐廳、咖啡廳、公園、車站等，顯示對應飾品 emoji。再按一次關閉。

⚠️ 需要網路，第一次可能要等幾秒。

📖 飾品圖鑑頁

所有 96 種飾品分類。點分類展開 → 點皮克敏圖示標記已收集（綠色邊框）→ 再點取消。

上方可篩選類型、搜尋名稱（中英文皆可）。

📊 統計頁

總收集進度 → 各分類收集狀態 → 格子標記統計。

📤 匯出備份 匯出 JSON 備份 ｜ 📥 匯入還原 匯入還原（換手機用）。

💡 建議定期匯出備份，清除瀏覽器資料會遺失。

🎮 實戰刷盆流程

出門前

  1. GPS 定位 → 開 POI 看附近飾品點
  2. 找有多個格子的大純點（公園、百貨公司）
  3. 用 ⭐ bookmark 標記想去的格子

到現場

  1. GPS 確認你在哪個格子
  2. 在格子裡走 1,000～3,000 步觸發掉落
  3. 刷完標記 ✅ farmed，CD 中標記 ⏳
  4. 移動到下一格繼續

回家後

  1. 統計頁看成果 → 圖鑑頁標記新飾品 → 匯出備份

❓ 常見問題

Q: 顯示「格子太多」？
放大地圖或降低 Level。

Q: POI 沒反應？
Overpass API 有速率限制，等幾秒再試。

Q: 換手機？
統計頁匯出備份 → 新手機匯入還原。

Q: 跟 Pik-Tool 差在哪？
我們不需要帳號，資料存手機本地。多了 S2 多 Level 切換和格子標記。
