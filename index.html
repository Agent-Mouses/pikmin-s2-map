📦 Open Source Credits

  • scott0127/pik_tool (MIT License) — Decor data (decor.json), OSM tag→decor mapping rules, Overpass API architecture. Original decor catalog and collection tracker for Pikmin Bloom.
  • jonatkins/s2-geometry-javascript — S2 Cell geometry calculations (face/ij/level, vertex computation, Hilbert curve). Ported to our s2.js module.
  • s2-geometry (npm) (MIT/Apache-2/ISC) — Reference for S2 Cell ID encoding and key conversion.
  • pixlpirate/pikmin-map (AGPL-3.0) — Inspiration for OSM-based Pikmin decor location mapping.
  • Leaflet.js (BSD-2-Clause) — Interactive map library.
  • OpenStreetMap (ODbL) — Map tile data.
  • Overpass API — OSM data querying for POI locations.
  • Google S2 Geometry — Original S2 Cell system documentation and algorithms.
